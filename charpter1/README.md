# 一、内存

**问 0001：内存需要被有序的进行管理，那么内存管理在干啥？**

从技术实现角度来看 [<sup>[1]</sup>](/reference/README.md/#ref0001)，Linux 内存管理的实现涵盖了如下领域。

- 如何管理物理内存？
    - 内存中的物理内存页的管理；
- 大块内存如何分配？
    - 伙伴系统；
- 较小块的内存和分配？
    - slab 分配器；
    - slub 分配器；
    - slob 分配器；
- 非连续内存块如何处理？
    - vmalloc 机制；
- 进程的地址空间
    - （未解决）还不知道为什么在这里会提及进程；
